<h2>JavaScript Features:</h2>
<ul>
    <li><strong> Form Handling:</strong>  Captures user input from forms (recipe/category) and validates it before submission, using addEventListener on forms for submission prevention and validation.</li>
    <li><strong>Image Preview:</strong>  Uses FileReader API to allow users to preview uploaded images before submitting the form.</li>
    <li><strong>Smooth Scroll:</strong>  Smoothly scrolls between sections of the page using either scroll-behavior: smooth (CSS) or JavaScript anchor navigation.</li>
    <li><strong>Interactive Navigation Bar:</strong>  Highlights the selected menu item or dynamically updates based on user interaction using event listeners.</li>
    <li><strong>Dynamic Recipe Display:</strong>  Adds/removes recipes in the DOM without reloading the page, using functions that append new elements to the recipe container.</li>
    <li><strong>Button Interaction:</strong>  Listens for button clicks (like viewing recipes) and triggers actions such as opening modals or navigating.</li>
    <li><strong>Animations:</strong>  Adds hover effects (like scaling or box-shadow) for elements such as .box using event listeners for smooth transitions.</li>
    <li><strong>AJAX/Fetch API:</strong>  Sends or fetches data asynchronously from the server, like adding new recipes or categories, without page reloads.</li>

</ul>

<h2>Modules Used: </h2>
<p>In this project, the following modules were used:</p>

<ul>
    <li><strong>Express</strong>: Handles routing, middleware, and HTTP requests.</li>
    <li><strong>Express-Session</strong>: Manages user sessions for authentication and session persistence.</li>
    <li><strong>MySQL (mysql2)</strong>: Provides a connection to a MySQL database for data storage and retrieval.</li>
    <li><strong>Path</strong>: Resolves file paths and serves static files.</li>
    <li><strong>Multer</strong>: Manages file uploads, processing multipart form data.</li>
</ul>

<p>These modules work together to create a dynamic web application with session management, database interactions, and file upload handling.</p>
